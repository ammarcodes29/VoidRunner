# Hand Landmarker Model

The CV mode requires the MediaPipe hand landmarker model file.

## Download Instructions

1. Download `hand_landmarker.task` from:
   https://storage.googleapis.com/mediapipe-models/hand_landmarker/hand_landmarker/float16/latest/hand_landmarker.task

2. Place the file in this directory (`voidrunner/assets/models/`)

## Alternative Download (Command Line)

```bash
cd voidrunner/assets/models
curl -O https://storage.googleapis.com/mediapipe-models/hand_landmarker/hand_landmarker/float16/latest/hand_landmarker.task
```

## File Size

The model file is approximately 12MB.
